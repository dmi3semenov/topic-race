# Topic Race — roadmap

## Reels / vertical format (9:16)

Цель: возможность экспортировать / показывать анимацию в вертикальном формате
Reels / TikTok / Shorts, чтобы:
- загружать в Instagram Reels и YouTube Shorts
- показывать коллегам на телефоне в удобном формате
- накладывать трек «Universe Size» (сравнение планет Солнечной системы) и выкладывать
  как полноценный контент

Что нужно сделать:
- [ ] Вертикальный layout D3: соотношение 9:16 (1080×1920 или 720×1280).
      Шрифты / отступы / размер даты подстроить под узкую ширину.
- [ ] Добавить в UI переключатель «Landscape / Reels (vertical)».
- [ ] Фон: варианты (светлый / тёмный / с фоновой картинкой галактики под трек).
- [ ] Рендер в MP4 без звука (screen-recorder через Playwright/Puppeteer + ffmpeg).
- [ ] Муксинг аудио-трека в финальный MP4 (ffmpeg `-i video -i audio -c:v copy -shortest`).
- [ ] Подгонка длительности анимации под длительность трека «Universe Size»
      (вычислить `frame_ms * n_frames ≈ track_duration`).
- [ ] Проверить, что топик-эмодзи и русские названия рендерятся в mp4 корректно
      (шрифты с кириллицей).

## Сервис для чужих групп — долгосрочная идея

Превратить topic-race в сервис, где любой пользователь может указать свою группу
(и дать доступ), а на выходе получить ровно такой же Reels-ролик. Минимальный
UX: «подключить Telegram → выбрать группу → получить 30-120 сек вертикальное
видео под готовый саундтрек».

Варианты авторизации:
- **User-client Telethon** (как у нас сейчас) — нужен api_id/hash и код из
  Telegram. Сложно для обычного пользователя, но даёт доступ к приватным
  группам.
- **Mini App в Telegram** — запускается внутри Telegram, авторизуется через
  Telegram Login Widget. Доступна только «публичная» часть данных.
- **Бот, добавленный в группу админом** — тянет историю с момента добавления.
  Самый массовый UX, но старые сообщения недоступны.

Технические риски / ограничения:
- Flood waits на стороне Telegram API (мы уже их видели на 83 топиках).
  Для масштабирования нужна очередь с воркерами и адаптивный backoff.
- Авторские права на трек «Universe Size» — для публичного сервиса нужен либо
  лицензированный аналог, либо библиотека треков без ограничений (Pixabay,
  YouTube Audio Library).
- Хранение персональных сообщений — GDPR. Надо либо не хранить (стрим →
  рендер → удалить), либо явное согласие пользователя.

## Брейншторм: какой ещё viral-контент из Telegram-данных

Идея верхнеуровневая — сделать небольшой набор «генераторов из твоего
Telegram», которые люди захотят залить в Reels/Stories, потому что это про
них и выглядит классно. Кандидаты:

- **Wrapped по году:** топ чатов / часов / дней активности / эмодзи / слов.
  Аналог Spotify Wrapped, вертикальные карточки.
- **Heatmap «когда ты в Telegram»:** по часам × дням недели, из личной
  истории. Анимация «просыпания» по неделям.
- **Граф общения:** круг, в центре — пользователь, рёбра — самые активные
  диалоги, толщина по числу сообщений. Hover — количество.
- **Эволюция стикерпаков / эмодзи:** какие стикеры/эмоджи вытесняли какие
  со временем.
- **Гонка эмодзи / слов в одном чате:** например, в большом рабочем канале —
  какие слова / темы входили в тренд, когда.
- **«Самые живучие темы»:** топ-10 топиков, которые не умирают месяцами
  (используя длину «жизни» топика — от первого до последнего сообщения).
- **Путь сообщения:** для forwarded-message цепочка «кто кого репостит»
  внутри круга каналов.
- **Slowest text:** самые длинные паузы между сообщениями в чате / топике,
  визуализация в виде пустых дней.

Ключевой принцип: картинка должна быть персональной (про «меня»), готовой
под вертикальный формат Reels / Stories, и с музыкой/ритмом, попадающим в
тему. Вирусность = «про меня + красиво + одним кликом + удобно поделиться».
